# 🌟 KeenKeeper — Keep Your Friendships Alive

**KeenKeeper** is a modern friendship management web application designed to help users stay connected with friends by tracking interactions, organizing communication history, and visualizing friendship activity.

---

## 📌 Overview

Built with a focus on user experience, **KeenKeeper** allows you to log calls, texts, and video chats. It transforms raw interaction data into a structured timeline and provides analytics insights through visual charts. The application is fully responsive and optimized for all screen sizes.

---



---

## ✨ Key Features

### 👥 Friend Management
* Dynamic display of friends from JSON datasets.
* Features profile images, custom tags, status indicators, and contact details.
* Responsive card-based layout.

### ⚡ Interaction Timeline
* Automatically logs **Call**, **Text**, and **Video** interactions.
* Each entry is saved with a precise timestamp.
* Displays history in an easy-to-read chronological timeline.

### 📊 Friendship Analytics
* Integrated **Pie Charts** (via Recharts) to visualize interaction frequency.
* Quickly see the distribution between calls, texts, and video chats.

### 📱 Fully Responsive
* **Mobile Devices** 📱: Optimized touch interface.
* **Tablets** 📟: Adaptive grid layouts.
* **Desktops** 💻: High-performance dashboard view.
